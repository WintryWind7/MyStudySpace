【题目7：学生信息管理系统（综合挑战）】

难度：★★★★☆

题目描述：
设计一个简单的学生信息管理系统，实现以下功能：
1. 添加学生信息
2. 显示所有学生信息
3. 按成绩排序
4. 查找学生（按学号）
5. 保存到文件
6. 从文件读取

要求：
1. 定义结构体存储学生信息（姓名、学号、3门课程成绩、平均分）
2. 使用函数实现各个功能
3. 使用文件保存和读取数据

结构体定义：
struct Student {
    char name[20];
    char id[15];
    double scores[3];  // 三门课成绩
    double average;    // 平均分
};

功能菜单：
1. 添加学生
2. 显示所有学生
3. 按平均分排序
4. 查找学生
5. 保存到文件
6. 从文件读取
0. 退出

提示：
- 使用switch语句实现菜单选择
- 文件操作使用fopen、fwrite、fread、fclose
- 排序可以使用冒泡排序或选择排序
- 查找使用线性查找
