# 实验总结

本次课程设计成功实现了一个基于Java和MySQL的员工管理信息系统，采用MVC三层架构设计，包含用户认证、员工信息管理和考勤管理三大核心功能模块。系统使用控制台界面进行交互，通过JDBC技术实现与MySQL数据库的连接和数据操作。数据库设计包含管理员表、员工信息表和考勤状态表三个核心表，通过外键约束保证数据完整性，实现了员工信息与考勤记录的自动关联和级联删除功能。

系统经过全面的功能测试，各模块运行稳定，测试结果完全符合预期。用户认证模块测试中，成功注册管理员账户（账号8080，密码8080），验证了数据库初始无此账户，注册后能够正常登录系统。员工信息管理模块针对工号8080的测试数据进行了完整的增删改查操作验证，添加员工信息后成功查询到该记录，修改操作将工号8080的其他信息全部更新为9090，数据准确变更，最后删除测试数据时系统正确执行级联删除，同步清除了考勤表中的关联记录。考勤管理模块测试中，初始状态下所有员工均为离岗状态，将工号E003、E004修改为在岗状态，E001保持离岗，系统准确记录状态变更并实时更新时间戳，查询功能能够正确显示各员工的当前考勤状态。

系统设计注重用户体验，在登录注册等多步骤操作中增加了返回上级功能（输入0返回），避免了操作失误导致的无限循环，界面布局合理，操作提示清晰，菜单层级分明。通过本次实践，深入理解了Java面向对象编程思想、数据库设计原则以及MVC架构模式的应用，掌握了JDBC数据库操作技术和SQL语句的编写，提升了系统分析与设计能力，为今后开发更复杂的信息管理系统奠定了坚实基础。整个开发过程从需求分析、数据库设计、代码实现到功能测试形成了完整的软件开发流程，加深了对软件工程方法的理解和应用。
