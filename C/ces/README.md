# C语言综合练习题

本目录包含一系列C语言练习题，从基础到进阶，帮助你巩固所学知识。

## 题目列表

### 基础题（Level 1）
- **题目1**: 数字统计 - 考察循环和条件判断
- **题目2**: 数组反转 - 考察数组操作
- **题目3**: 字符串回文判断 - 考察字符串处理

### 进阶题（Level 2）
- **题目4**: 矩阵转置 - 考察二维数组
- **题目5**: 字符串排序 - 考察字符串数组和排序
- **题目6**: 成绩管理 - 考察结构体和函数

### 挑战题（Level 3）
- **题目7**: 学生信息管理系统 - 综合考察结构体、指针、文件操作
- **题目8**: 矩阵运算 - 考察指针、二维数组和函数

## 使用说明

1. 每个题目都有题目描述文件（题目X_题目.txt）
2. 你需要创建对应的.c文件来完成题目
3. 可以参考答案文件（题目X_答案.c），但建议先自己尝试
4. 编译运行测试你的代码

## 学习建议

- 按顺序完成题目，从基础到进阶
- 遇到困难先思考，再查看提示
- 完成后对比答案，学习更优的写法
- 尝试优化你的代码
