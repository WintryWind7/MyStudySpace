【题目5：字符串排序】

难度：★★★☆☆

题目描述：
输入n个字符串，按字典序（从小到大）排序后输出。

要求：
1. 定义函数 void SortStrings(char str[][50], int n)
2. 使用冒泡排序或选择排序
3. 使用strcmp()函数比较字符串

输入格式：
第一行：整数n（n <= 20）
接下来n行：每行一个字符串（长度不超过50）

输出格式：
排序后的字符串，每行一个

输入样例：
5
banana
apple
cherry
date
elderberry

输出样例：
apple
banana
cherry
date
elderberry

提示：
- 使用 <string.h> 中的 strcmp() 比较字符串
- strcmp(s1, s2) < 0 表示 s1 < s2
- 交换字符串需要使用 strcpy() 和临时变量
